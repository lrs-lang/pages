Function lrs::thread::cpus

Returns the CPU mask of a thread.

Syntax

fn cpus(thread: i32, buf: &mut [u8]) -> Result<&mut CpuMask, Errno>

Arguments

NameDescription
thread

The thread to inspect or 0 for this thread.

buf

The buffer in which the mask will be stored.
