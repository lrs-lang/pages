Constant lrs::file::mode::MODE_DIRECTORY

A mode with the default bits for a directory:

  • owner readable

  • owner writable

  • owner executable

  • group readable

  • group executable

  • world readable

  • world executable

Syntax

const MODE_DIRECTORY: Mode = Mode(S_IRUSR | S_IWUSR | S_IXUSR | S_IRGRP | S_IXGRP | S_IROTH | S_IXOTH);
