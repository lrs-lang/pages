Method lrs::group::Info::from_group_id

Retrieves group info of the group with a certain id.

Syntax

impl<'a> Info<'a> {
    fn from_group_id(buf: &'a mut [u8], id: u32) -> Result<Info<'a>, Errno>
}

Arguments

NameDescription
buf

The buffer in which the information will be stored.

id

The group id.
