Function lrs::thread::set_process_niceness

Sets the niceness of a process.

Syntax

fn set_process_niceness(id: i32, niceness: u8) -> Result<(), Errno>

Arguments

NameDescription
id

The process to modify, or 0 for this process.

niceness

The new niceness.

Remarks

The niceness is a value between 1 and 40, with higher values meaning more favorable scheduling.
