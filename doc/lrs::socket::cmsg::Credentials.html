Struct lrs::socket::cmsg::Credentials

Process credentials.

Syntax

struct Credentials {
    process_id: i32,
    user_id: u32,
    group_id: u32,
}

Fields

Remarks

Privileged process can send any ids. Unprivileged processes must send their process id and the other ids must be either the effective, real, or saved ids.
