Method lrs::socket::Socket::ipv6_raw

Creates a raw Ipv6 socket.

Syntax

impl Socket {
    fn ipv6_raw(proto: Proto, flags: SockFlags) -> Result<Socket, Errno>
}

Arguments

NameDescription
proto
flags

Remarks

This is equivalent to

Socket::new(domain::Ipv6, kind::Raw, proto.0 as c_int, flags)
