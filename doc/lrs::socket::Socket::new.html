Method lrs::socket::Socket::new

Creates a new socket.

Syntax

impl Socket {

}

Arguments

NameDescription

Remarks

This is the most general constructor. There are simpler, more specialized constructors for the common cases of Unix, Ipv4, and Ipv6 sockets.
