Method lrs::user::Info::find_by

Finds the first user that satisfies a predicate.

Syntax

impl<'a> Info<'a> {
    fn find_by<F>(buf: &'a mut [u8], pred: F) -> Result<Info<'a>, Errno>
        where F: Fn(&Info) -> bool,
}

Arguments

NameDescription
buf

The buffer in which the information will be stored.

The predicate.
