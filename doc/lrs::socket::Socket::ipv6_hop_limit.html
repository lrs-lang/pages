Method lrs::socket::Socket::ipv6_hop_limit

Retrieves the hop limit of packets sent over this socket.

Syntax

impl Socket {
    fn ipv6_hop_limit(&self) -> Result<Option<u8>, Errno>
}

Return value

Returns the hop limit of packets sent over this socket.
