Function lrs::syscall::lseek

Seeks in a file descriptor.

Syntax

fn lseek(fd: i32, offset: i64, whence: u32) -> i64

Arguments

NameDescription
fd

The file descriptor in which to seek.

offset

The range to seek.

whence

How to seek.

Return value

Returns the new position in the file descriptor or an error value.
