Struct lrs::msg_queue::MqAttr

Attributes of a message queue.

Syntax

struct MqAttr {
    flags: MqFlags,
    max_msgs: usize,
    cur_msgs: usize,
    msg_size: usize,
}

Fields

NameDescription
flags

Flags of the queue.

max_msgs

Maximum number of messages stored in the queue.

cur_msgs

Number of currently stored messages.

msg_size

The maximum size of a message in the queue.
