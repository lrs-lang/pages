Function lrs::syscall::setsockopt

Sets a socket option.

Syntax

fn setsockopt(sockfd: i32, level: i32, optname: i32, optval: &[u8]) -> i32

Arguments

NameDescription
sockfd

The socket.

level

The level of the option.

optname

The name of the option.

optval

The value to set.

Return value

Returns success or an error value.
