Method lrs::socket::Socket::ipv6_into_ipv4

Turns an Ipv6 socket into an Ipv4 socket.

Syntax

impl Socket {
    fn ipv6_into_ipv4(&self) -> Result<(), Errno>
}

Remarks

This only works if the socket is connected to and bound to Ipv6-mapped Ipv4 addresses.
