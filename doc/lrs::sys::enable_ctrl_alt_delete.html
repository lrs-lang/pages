Function lrs::sys::enable_ctrl_alt_delete

Enable or disable immediate restarting with ctrl-alt-delete.

Syntax

fn enable_ctrl_alt_delete(enabled: bool) -> Result<(), Errno>

Arguments

NameDescription
enabled

Remarks

If enabled, pressing ctrl-alt-delete immediately restarts the system. This can cause data-loss.

If disabled, pressing ctrl-alt-delete sends SIGINT to init which then decides how to proceed.
