Struct lrs::socket::Domain

A socket domain/protocol family

Syntax

struct Domain(i32)

Fields

PositionDescription
1

The integer constant associated with the domain.

Trait implementations

NameDescription
Clone

Objects that can be duplicated.

Copy

Objects that can safely be copied via memcpy.

Debug

Objects that can be formatted in a "debug" form.

Eq

Objects that implement the binary == and != operators.

MaybeClone

Objects that can be duplicated.

Pod

Objects that are safe to use if they contain a random bit pattern.

Remarks

This is a protocol family understood by the kernel. For direct access to the link layer (TCP/IP model) use the Packet domain. See also packet(7).

See lrs::socket::domain for pre-defined constants.
