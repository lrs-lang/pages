Method lrs::msg_queue::MsgQueue::recv

Receives a message over the message queue.

Syntax

impl MsgQueue {
    fn recv(&self, buf: &'a mut [u8]) -> Result<(&'a mut [u8], u16), Errno>
}

Arguments

NameDescription
buf

A buffer in which the message will be stored.

Return value

Returns the message and its priority.
