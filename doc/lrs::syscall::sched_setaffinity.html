Function lrs::syscall::sched_setaffinity

Sets a thread's CPU affinity mask.

Syntax

fn sched_setaffinity(tid: i32, set: &[u8]) -> i32

Arguments

NameDescription
tid

The id of the thread.

set

The cpu affinity of the thread.

Return value

Returns success or an error value.
