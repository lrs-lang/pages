Function lrs::file::real_path

Returns a canonicalized absolute path.

Syntax

fn real_path<P>(path: P) -> Result<CString<Libc>, Errno>
    where P: ToCString,

Arguments

NameDescription
path

The path to canonicalize.

Remarks

The path will not contain any /./, /../, or //.
