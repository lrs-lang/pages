Method lrs::pipe::Pipe::new

Creates a new pipe.

Syntax

impl Pipe {
    fn new(flags: PipeFlags) -> Result<(Pipe, Pipe), Errno>
}

Arguments

NameDescription
flags

Flags to use when creating the pipe.

Return value

Returns the write end (first element) and the read end (second argument) of the pipe.
