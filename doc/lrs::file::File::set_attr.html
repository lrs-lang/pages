Method lrs::file::File::set_attr

Sets an attribute of this file.

Syntax

impl File {
    fn set_attr<S, V>(&self, name: S, val: V) -> Result<(), Errno>
        where S: ToCString,
              V: AsRef<[u8]>,
}

Arguments

NameDescription
name

The name of the attribute.

val

The new value of the attribute.
