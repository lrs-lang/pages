Method lrs::vec::Vec::push_all

Appends a slice of copyable elements to the vector.

Syntax

impl<T, H> Vec<T, H>
    where H: Allocator,
{
    fn push_all(&mut self, vals: &[T]) -> Result<(), Errno>
        where T: Copy,
}

Arguments

NameDescription
vals

The elements to append.

Remarks

If this operation fails, no elements have been appended.
