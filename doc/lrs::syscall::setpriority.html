Function lrs::syscall::setpriority

Sets the scheduling priority of a process, process group, or user.

Syntax

fn setpriority(which: i32, who: i32, prio: i32) -> i32

Arguments

NameDescription
which

The id type of who.

who

Whose priority to set.

prio

The scheduling priority.
