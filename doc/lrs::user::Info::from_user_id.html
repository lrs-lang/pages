Method lrs::user::Info::from_user_id

Retrieves user info of the user with a certain id.

Syntax

impl<'a> Info<'a> {
    fn from_user_id(buf: &'a mut [u8], id: u32) -> Result<Info<'a>, Errno>
}

Arguments

NameDescription
buf

The buffer in which the information will be stored.

id

The id of the user.
